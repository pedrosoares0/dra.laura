# Dra. Laura Laytynher — Landing Page
🔗 [Acesse aqui](https://dralaura.netlify.app/)  
⚠️ **Melhor visualização no mobile – responsividade ainda em desenvolvimento.**

> **Um espaço digital que reflete a essência de um atendimento humanizado e de excelência.**

Este projeto foi desenvolvido para ser o cartão de visitas digital da Dra. Laura Laytynher, Cirurgiã-Dentista especializada em odontologia estética e restauradora. O objetivo principal foi criar uma experiência de navegação **fluida, elegante e acolhedora**, transmitindo confiança e profissionalismo logo no primeiro contato.

A interface foi pensada para destacar a transformação de sorrisos com naturalidade, facilitando o acesso às informações sobre tratamentos e o agendamento de consultas via WhatsApp.

---

## ✨ Destaques do Projeto

*   **Design Minimalista & Premium:** Uso de tipografia refinada, paleta de cores suaves e elementos visuais que remetem à sofisticação e cuidado.
*   **Experiência Mobile-First:** Focado totalmente na usabilidade em celulares, garantindo que o paciente tenha a melhor experiência na palma da mão.
*   **Animações Sutis:** Elementos que surgem suavemente ao rolar a página (efeito *reveal*), criando uma narrativa visual envolvente.
*   **Agendamento Simplificado:** Botões de ação estrategicamente posicionados que levam diretamente para uma conversa personalizada no WhatsApp.

## 👩‍⚕️ Sobre a Profissional

A Dra. Laura Laytynher atua com foco em:
*   **Odontologia Restauradora:** Devolvendo função e estética.
*   **Lentes em Resina Composta:** Sorrisos renovados com naturalidade.
*   **Prótese Fixa e Removível:** Reabilitação completa e segura.

<summary><strong>🛠️ Informações Técnicas (Para Desenvolvedores)</strong></summary>
<br>

Este projeto foi construído utilizando tecnologias modernas de desenvolvimento web focado em alta performance, semântica e animações fluidas de nível premium.

### Tecnologias Utilizadas
*   **React (TypeScript):** Componentização e controle de estados robustos.
*   **Vite:** Tooling ultrarrápido para desenvolvimento e empacotamento otimizado de produção.
*   **Tailwind CSS:** Estilização modular com suporte integrado a temas e variáveis de CSS nativas para alternância de modo Safira/Pérola.
*   **GSAP (GreenSock) + ScrollTrigger:** Animações premium com alta taxa de quadros e efeitos de revelação por desfoque e cortina baseados no scroll.
*   **Framer Motion:** Micro-interações leves e fluidas.

### Como Rodar o Projeto Localmente

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/pedrosoares0/dra.laura.git
    ```
2.  **Acesse a pasta do projeto:**
    ```bash
    cd dra.laura
    ```
3.  **Instale as dependências:**
    ```bash
    npm install
    ```
4.  **Execute o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```
